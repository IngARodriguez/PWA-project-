3. Como Usar en tu PWA

3.1 Importar los estilos

En tu App.tsx, agrega estos imports:

import './theme/nequi-design-system.css';  // Design tokens
import './theme/nequi-components.css';      // Clases de componentes

3.2 Usar variables CSS

/* En cualquier CSS */
.mi-boton {
  background-color: var(--nq-color-primary);
  border-radius: var(--nq-radius-xl);
  padding: var(--nq-space-4) var(--nq-space-6);
  font-size: var(--nq-text-lg);
}

3.3 Usar clases de componentes

<!-- Boton principal -->
<button className="nq-btn nq-btn-primary nq-btn-block">Entra</button>

<!-- Card -->
<div className="nq-card nq-card-body">Contenido</div>

<!-- Chip de estado -->
<span className="nq-chip nq-chip-success">Abierto</span>

<!-- Chat bubble -->
<div className="nq-chat-bubble-agent">Hola!</div>
<div className="nq-chat-bubble-user">Hola!</div>

<!-- Typing indicator -->
<div className="nq-typing-indicator">
  <div className="nq-typing-dot"></div>
  <div className="nq-typing-dot"></div>
  <div className="nq-typing-dot"></div>
</div>

3.4 Usar iconos SVG

<!-- Importar sprite de iconos una vez -->
import icons from '../assets/icons/nequi-icons.svg';

<!-- Usar un icono -->
<svg width="24" height="24">
  <use href={`${icons}#nq-icon-back`} />
</svg>
